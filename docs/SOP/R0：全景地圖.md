## R5：Cloud 光源重構 ❌【2026-04-26 撤回】

- **撤回原因**：使用者原以為「減光源（4→1）」可大幅加速降噪到 C1 / C2 等級。實驗（CONFIG 5 整面光板）驗證後物理結論為：收斂瓶頸不在「光源數量」、在「下一次事件估計（NEE）採樣效率」。C3 / C5 同屬「朝外光源」、對下方 hit 點 cosLight ≈ 0、NEE 採樣 50%+ 浪費 → 高變異 → 髒、是物理本質限制非實作 bug
- **實驗 archive**：worktree `Home_Studio_3D-r5-panel-config5/` 完整保留 fix02 ~ fix05-maxch 6 輪迭代 + .omc/ PRD 紀錄
- **量化結論**：fix04-cdf 螢火蟲 27 → 0 px、亮度差 −1.11%、色比差 −3.55%（max-channel normalize 修法後）
- **詳見**：`R5：Cloud光源重構.md` §R5-Z 結論段
